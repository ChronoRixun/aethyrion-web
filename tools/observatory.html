Get Started in 5 Minutes

Deploy with Docker or install locally

# Option 1: Docker Compose (Easiest)
git clone https://github.com/aethyrion/observatory.git
cd observatory
docker-compose up -d

# Observatory is now running!
# API: http://localhost:3101
# OTLP Collector: http://localhost:3102

# Option 2: Local Development
npm install
npm run setup
npm run dev

Instrument Your Apps

Works with any OpenTelemetry-compatible instrumentation

// Node.js / Next.js
npm install @opentelemetry/sdk-node @opentelemetry/auto-instrumentations-node

import { NodeSDK } from '@opentelemetry/sdk-node';
import { OTLPTraceExporter } from '@opentelemetry/exporter-trace-otlp-http';

const sdk = new NodeSDK({
  serviceName: 'your-app',
  traceExporter: new OTLPTraceExporter({
    url: 'http://localhost:3102/v1/traces'
  })
});

sdk.start(); // That's it!
